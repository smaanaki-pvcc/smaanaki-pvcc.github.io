Operating Systems Terminology
TERM DEFINITION
operating system The software that controls the overall operation of the computer including file management, the user interface, and the program execution environment.
job queue A data structure that keeps track of the jobs to be processed by a computer system.
FIFO An algorithm for scheduling tasks where the first task to arrive is the first to be processed.
interactive processing A method where the user interacts with a program while it's running, providing inputs which the program processes immediately.
time-sharing A method of operating system process scheduling where multiple users can access the system concurrently, with the system's time divided into short intervals for each user.
multiprogramming A method that allows multiple programs to reside in separate partitions of memory at the same time and be executed concurrently.
application software Consists of the programs for performing tasks particular to the machineâ€™s utilization
system software Software designed to manage the system resources and provide a platform for application software.
utility software System management tools and programs designed to assist in the maintenance and care of a computer.
graphical user interface (GUI) A user interface that allows users to interact with electronic devices through graphical icons and visual indicators.
shell A user interface for access to an operating system's services, which can be command-line (CLI) or graphical (GUI).
kernel The core part of an operating system, managing system resources and the communication between hardware and software components.
device driver Software that tells the operating system how to interact with hardware components.
file manager A program that provides a user interface to manage files and folders.
memory manager A part of the operating system that handles the allocation of memory to various applications.
virtual memory A memory management capability of an OS that uses hardware and software to allow a computer to compensate for physical memory shortages, temporarily transferring data from random access memory (RAM) to disk storage.
boot strapping The process of starting a computer, initiating its hardware and software (OS).
ROM (Read-Only Memory) A type of non-volatile storage that is typically used to store firmware or application software in plug-in cartridges.
deadlock A situation in concurrent programming where two or more processes are unable to proceed because each is waiting for the other to release resources.
spooling A process of sending data to a spool, or temporary storage area, from which it can be read and processed at a later time, commonly used in printing.
super user (or administrator) A user with full access to all the files and commands on a system, typically used for system administration tasks.
multi-factor authentication (MFA) A security system that requires more than one method of authentication from independent categories of credentials to verify the user's identity for a login or other transaction.
