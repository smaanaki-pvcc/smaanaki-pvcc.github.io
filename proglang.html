Programming Languages

Term Definition
Machine Language The lowest-level programming language, directly understood by the computer's CPU.
Debugging The process of identifying and removing errors from computer software.
Assembler A program that converts assembly language code into machine language.
Assembly Language A low-level programming language that is a symbolic representation of machine code.
Translator A program that converts code from one programming language to another.
Compiler A program that translates source code written in a high-level programming language into machine code.
Interpreter A program that executes instructions written in a programming or scripting language without requiring them to be compiled.
Cross-platform Software Software designed to work on multiple computer platforms or operating systems.
Natural Language Any language used by humans to communicate, such as English or Spanish.
Formal Language A set of strings of symbols constrained by specific grammatical rules.
Machine Independence The characteristic of software that allows it to run on various types of hardware without modification.
High-level Programming Language A programming language with strong abstraction from the details of the computer.
Object-oriented Programming (OOP) A programming paradigm based on the concept of objects, which contain data and methods.
Scripting Language A programming language designed for integrating and communicating with other programming languages.
Data Types Categories of data that determine the type of value a variable can hold, such as integer, float, and string.
Interger A data type representing whole numbers.
Float A data type representing numbers with fractional parts.
String A data type representing sequences of characters.
Primitive Data Type Basic data types provided by a programming language as building blocks.
Data Structure A particular way of organizing data in a computer so that it can be used effectively.
Python Collection Data Structures Data structures available in Python that include lists, tuples, sets, and dictionaries.
List An ordered collection of items that can be changed.
Tuple An ordered collection of items that cannot be changed.
Set An unordered collection of unique items.
Dictionary A collection of key-value pairs, with each key being unique.
Function A block of organized, reusable code that performs a single action.
Local Variable A variable that is declared inside a function and is only accessible within that function.
Global Variable A variable that is declared outside of all functions and is accessible throughout the program.
Source Program The original program written in a high-level programming language.
Object Program The output of a compiler after it translates the source program.
