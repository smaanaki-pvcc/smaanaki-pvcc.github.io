Chapter 4: Networking Terms and Definitions
TERM DEFINITION
Network Classifications
  • LAN: Local Area Network, a network that spans a relatively small space.
  • MAN: Metropolitan Area Network, a network that covers a larger geographic area than LAN, spanning multiple buildings or even a whole city.
  • WAN: Wide Area Network, a network that covers broad area such as multiple cities, countries, or continents.
Network models
  • Bus network: A network topology where each node is directly connected to a single communication line.
  • Star network: A network topology with one central hub to which all other nodes are connected.
Protocols
  • CSMA/CD Ethernet: A network access method used on shared network topologies to control data transmission and avoid collisions.
  • CSMA/CA Wifi: A protocol for carrier transmission that tries to avoid collisions by broadcasting intent before transmitting.
Combining networks
  • Repeater: A network device that regenerates or replicates signals to extend the transmission distance.
  • Bridge: A network device that connects multiple network segments at the data link layer.
  • Switch: A networking device that filters and forwards packets between LAN segments.
  • Router: A device that forwards data packets along networks, typically used to connect networks and route traffic between them.
  • Gateway: A node that serves as an access point to another network, often involving different protocols.
Client/server model
  • P2P (peer-to-peer): A distributed application architecture that partitions tasks or workloads between peers.
  • Distributed system: A system whose components are located on different networked computers which communicate and coordinate their actions by passing messages.
  • Cloud computing: The delivery of computing services—including servers, storage, databases, networking, software, analytics, and intelligence—over the internet (“the cloud”).
Internet Addressing
  • Tier 1: Internet Service Providers (ISPs) that can reach every network on the internet without paying transit fees to another ISP.
  • Tier 2: ISPs that typically obtain internet transit at a cost from Tier 1 networks to reach the broader internet.
  • Tier 3: ISPs that rely on higher-tier ISPs to access the entire internet, usually customer-oriented ISPs providing local residential internet access.
IP Address
  • ICANN: The organization that coordinates the allocation and assignment of unique identifiers for the internet, ensuring stable and secure operation. More about ICANN
  • DNS: Domain Name System, the phonebook of the internet that translates domain names to IP addresses so browsers can load internet resources.
  • TLD: Top-Level Domain, the last segment of the domain name, following the final dot of any domain name. List of Country-code TLDs
Internet protocols
  • FTP: File Transfer Protocol, a standard network protocol used for the transfer of computer files between a client and server on a computer network.
  • HTTP: Hypertext Transfer Protocol, the foundation of data communication for the World Wide Web.
  • SMTP: Simple Mail Transfer Protocol, an internet standard for email transmission across IP networks.
World Wide Web
  • URL: Uniform Resource Locator, the address of a web resource on the internet.
  • Client-side activities: Actions that occur on the client's computer, such as web browsing and running client-side scripts.
  • Server-side activities: Server operations that occur, such as serving web pages or running server-side scripts.
Internet four-layer software
  • Application layer: Provides protocols which directly serve the user's applications, such as HTTP for web browsing.
  • Transport layer: Provides communication session management between host computers, such as TCP and UDP.
  • Network layer: Deals with routing of packets through the network, such as IP protocol.
  • Link layer: Provides data transfer between adjacent network nodes and handles error detection and framing.
Internet is
  • Packet-switching network: A method of grouping data that is transmitted over a digital network into packets.
  • Fault tolerant: The ability of a system to continue operation even if a component fails.
  • Redundant: The duplication of critical components of a system with the intention of increasing reliability.
Cybersecurity
  • Malware: Software designed to disrupt, damage, or gain unauthorized access to a computer system.
  • Virus: A type of malware that replicates by inserting copies of itself into other computer programs.
  • Worm: A malware program that replicates itself in order to spread to other computers.
  • Trojan horse: Any malicious computer program which misleads users of its true intent.
  • Spyware: Software that enables a user to obtain covert information about another's computer activities.
  • Phishing: The attempt to obtain sensitive information such as usernames, passwords, by disguising as a trustworthy entity.
  • DDoS attack: Distributed Denial of Service attack, an attempt to make an online service unavailable by overwhelming it with traffic from multiple sources.
Protection
  • Firewall: A network security device that monitors incoming and outgoing network traffic and decides whether to allow or block specific traffic.
  • Spam filter: A program that is used to detect unsolicited and unwanted email and prevent those messages from getting to a user's inbox.
  • Proxy server: A server that acts as an intermediary for requests from clients seeking resources from other servers.
  • Antivirus software: Software designed to detect and destroy computer viruses.
  • Cryptography: The practice and study of techniques for secure communication in the presence of third parties called adversaries.
  • Public key encryption: A cryptographic system that uses pairs of keys: public keys which may be disseminated widely, and private keys which are known only to the owner.
